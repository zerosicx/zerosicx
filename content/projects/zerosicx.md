A pixel-art, game-like personal website where visitors navigate a top-down room inspired by classic Pokémon games and Stardew Valley.

Built with Phaser 3 for the game layer and Tailwind CSS for the UI overlays. Content is written in markdown and parsed at build time — no CMS required.
